# 🚀 Uber RideShare Data Analysis using SQL

## 📌 Objective
Analyze a ride-sharing dataset to uncover insights related to revenue, demand patterns, driver performance, and rider behavior.

---

## 📊 Dataset Overview
The dataset includes:
- Users (riders & drivers)
- Trips
- Payments
- Locations
- Reviews & cancellations

---

## 🔍 Key Insights

- 💰 Total revenue generated: **$604K (~₹5.0 Cr)**
- 🌆 Houston generates highest revenue (~$188K)
- 📈 Peak demand occurs at **8 AM, 2 PM, and 6–8 PM**
- ⚡ Surge pricing increases average fare from **$28 → $59 (~2x)**
- 🔁 **75% riders are repeat users**
- ❌ Cancellation rate is **~14–15% across cities**
- 📍 High demand-supply gaps identified in multiple zones
